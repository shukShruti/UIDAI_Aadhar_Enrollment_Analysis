# UIDAI_Aadhar_Enrollment_Analysis 
DRIVE LINK :- https://drive.google.com/file/d/1RDDxmR6E_WI-bcK9xaj2hPp4JoXirFAO/view?usp=sharing

Hackathon project under the Government of India, focused on analyzing Aadhaar enrollment datasets from UIDAI. This repository showcases data-driven insights, dashboards, and visualizations to explore citizen registration trends and improve process efficiency.

A comprehensive **data analytics and visualization project** that analyzes Aadhaar enrollment patterns across Indian states and districts with a special focus on **child enrollment trends, growth rates, efficiency metrics, and geographic performance insights**.

This project uses **Python, Pandas, NumPy, Matplotlib, and Seaborn** to process raw enrollment data and generate a **high-quality professional dashboard** for reporting, presentations, and hackathon or academic submissions.

---

## 🚀 Features

- 📍 **State-wise and District-wise Analysis**
- 👶 **Child vs Adult Enrollment Distribution**
- 📈 **Daily & Weekly Enrollment Trends**
- 🗺️ **Geographic Coverage & Efficiency Metrics**
- 📊 **Top Performing States Comparison**
- 🔥 **Growth Rate & Moving Average Tracking**
- 📉 **Outlier Detection & Statistical Summary**
- 🖼️ **High-Resolution Exportable Dashboard (PNG)**

To build a **data-driven intelligence system** that helps:

## 📥 Dataset Format

Your CSV file should contain at least the following columns:

| Column Name | Description |
|------------|------------|
| date | Enrollment date |
| state | State name |
| district | District name |
| pincode | Area pincode |
| age_0_5 | Enrollments (0–5 years) |
| age_5_17 | Enrollments (5–17 years) |
| age_18_greater | Enrollments (18+ years) |

> The script automatically cleans and standardizes column names and state formats.
